# Changelog

## Release 7.41.1

### Changes

* Shynixn/BlockBall#527 Replacing the entire tennis ball physics engine with the new 2026 tennis ball engine.
* Shynixn/BlockBall#527 All information can be found on this new [wiki page](https://shynixn.github.io/MCTennis/wiki/site/ball/)
* Shynixn/BlockBall#527 The tennis ball is no longer tied to the arena and can be configured in the new ``plugins/MCTennis/ball`` folder.
* Shynixn/BlockBall#527 Added new tennis ball properties and replaced existing ones to make it more realistic.
* Shynixn/BlockBall#527 The tennis ball can now be configured to have a realistic bounce, spin, and friction.
* Shynixn/BlockBall#527 The tennis ball now bounces off walls and players in a more realistic way.
* Shynixn/BlockBall#527 Added configurable tennis ball interactions. You can now configure to take sprinting, sneaking, selected hotbar slots, and more into account when interacting with the tennis ball. e.g. the tennis ball can be shot further while sprinting, the tennis ball can do a left curve when the left slots are selected
* Shynixn/BlockBall#527 The [developer api](https://shynixn.github.io/MCTennis/wiki/site/api/) has changed. You can now spawn tennis balls outside of games and use it for other minigames or purposes on your server.
* Shynixn/BlockBall#744 Added support for Minecraft 26.2
* Shynixn/BlockBall#743 Added a new flag to allow or deny block and items interactions during MCTennis in Minigame mode.
